¿Tengo una cuenta de Pearson?

Si alguna vez utilizó un producto de Pearson, probablemente tenga una cuenta. Le recomendamos utilizar una cuenta existente en lugar de crear una nueva. Esto le permitirá ahorrar tiempo y, además, solo tendrá un nombre de usuario y una contraseña para recordar.

Para obtener la información de su cuenta, seleccione ¿Olvidó su nombre de usuario o contraseña? y siga las instrucciones. Si tiene una cuenta, recibirá un correo electrónico con su nombre de usuario y un vínculo para restablecer su contraseña. (Si vinculará cuentas desde un Sistema de administración de aprendizaje, la página Restablecer contraseña se abrirá en una ventana o pestaña nuevas. Después de restablecer su contraseña, cierre ambas ventanas o pestañas, y vuelva a su Sistema de administración de aprendizaje para continuar).

El nombre de usuario y la contraseña distinguen mayúsculas de minúsculas.
