Que se passe-t-il une fois que je demande mon nom d'utilisateur et mon mot de passe ?

Si vous avez un compte Pearson, vous recevrez un e-mail contenant votre nom d'utilisateur ainsi qu'un lien permettant de réinitialiser votre mot de passe. Si vous avez plusieurs comptes, l'e-mail comportera une liste de tous vos noms d'utilisateur ainsi que différents liens permettant de réinitialiser vos mots de passe.

Si vous ne recevez pas notre e-mail : suivez les conseils indiqués dans le message que vous obtenez après votre demande de réinitialisation de mot de passe.

Remarque : si vous liez des comptes à partir d'un système de gestion de l'apprentissage, la page de réinitialisation du mot de passe s'ouvrira dans une nouvelle fenêtre ou un nouvel onglet. Après la réinitialisation du mot de passe, fermez les fenêtres ou les onglets et revenez à votre système de gestion de l'apprentissage si vous voulez continuer.
