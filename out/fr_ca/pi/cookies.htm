Comment puis-je activer les témoins (cookies)?

Les témoins sont des fichiers provenant de sites Web qui stockent les informations de navigation, y compris les préférences et les données de profil. Pour vous connecter ou lier des comptes, vous devez activer les témoins de sites tierces.

  • Chrome et Firefox : Autorisez tous les témoins ou ajoutez une exception pour pi.pearsoned.com.
  • Internet Explorer : Autorisez tous les témoins ou ajoutez des exceptions pour pi.pearsoned.com et pearson.com.
  • Safari et Edge : Autorisez tous les témoins.

Si vous rencontrez des difficultés, tentez de supprimer les fichiers de la mémoire cache du navigateur.

Consultez l’aide de votre navigateur si celui-ci n’est pas traité par les présentes directives.
