Search Tips

You can search for Pearson content using specific search terms such as a title, description, ISBN, or author name. You can broaden your search using other keywords such as introduction or psychology.

Search types

Your search term can contain a keyword or phrase.

Keyword

A keyword is a single word without quotes. For example, psychology returns all results with that keyword such as Beginning Psychology and Introduction to Psychology.

Phrase

A phrase is a group of words enclosed with double quotes. For example, "beginning psychology" only returns results containing the entire phrase, such as Beginning Psychology for Social Workers, but not Introduction to Psychology.

If you search for a phrase without double quotes, each word in the search terms is considered a separate keyword. For example, beginning psychology returns items including beginning or psychology, such as Beginning Algebra and Applied Psychology.

Wildcard searches with partial keywords

You can search with a wildcard (* or ?) representing one or more characters.

Note: You cannot use a wildcard as the first character of a search.

Single-character wildcard

For a single-character wildcard, enter a question mark (?). For example, te?t returns test or text.

Multiple-character wildcard

For a multiple-character wildcard, enter an asterisk (*). For example, test* returns test, tests, or tester.

You can also use a wildcard in the middle of a search term. For example: te*t

Combine or exclude keywords (OR, AND, NOT)

You can use OR, AND, or NOT (also called Boolean operators) to broaden or narrow your search.

Note: Always enter Boolean operators in all uppercase letters.

OR

If you use OR in a search term, your results include either keyword. If there is no Boolean operator between two search terms, OR is the default. For example, beginning OR psychologyand beginning psychology return Beginning Psychology and Beginning Sociology.

You can also use the symbolic equivalent of OR, which is &&

AND

If you use AND in a search term, your results only include items with both keywords. For example, beginning AND psychology returns Beginning Psychology and Beginning Psychology for Social Workers but not Beginning Sociology.

You can also use the symbolic equivalent of AND, which is ||

NOT

If you use NOT in a search term, your results exclude the keyword. For example, psychology NOT beginning returns Introduction to Psychology and Applied Psychology but not Beginning Psychology.

You can also use the symbolic equivalent of NOT, which is !
