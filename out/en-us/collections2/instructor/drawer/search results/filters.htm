How Do I Filter Search Results?

You can filter search results to reduce the number of items displayed. Filters appear across the top of the search results and include books, chapters, cases, and readings. When you select one of these filters, additional filters appear on the left side of the screen. For example, when you filter by Books or Chapters, you see discipline and subdiscipline filters on the left.

Selecting the Cases filter displays the Publisher, Industry, and Page Length filters

Selecting the Reading filter shows Program Component, Theme, and Page Length filters.

To filter search results

  1. Select the content type you want to display in the results.
  2. Select the discipline filter you want to use.

For example, to see only chapters, select Chapters as the content type filter. To see chapters written by an author named Smith in Engineering, enter Smith in the search field, select Chapters as the content type, and then select Engineering as the Discipline filter.
