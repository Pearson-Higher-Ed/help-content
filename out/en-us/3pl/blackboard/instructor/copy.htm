How do I share Pearson content with other instructors?

You can use a code to share your content with other instructors.

To generate a code:

  1. Select Allow copy from the options menu to the right of your content details.
  2. Turn on Allow this content to be copied by another instructor.
  3. Select Copy code next to the code that appears.
  4. Give the code to other instructors, for example, by pasting it into an email.

To use an existing code:

  1. Under Copy another instructor's content, enter or paste the code and select Find.
  2. Select Set it up to add the copied content's display name and availability dates.
