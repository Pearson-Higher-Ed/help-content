How do I enable cookies?

Cookies are files from websites that store your browsing information, including preferences and profile. To sign in or link accounts, you need to enable third-party cookies from pi.pearsoned.com.

Tip: If you already enabled cookies and have trouble signing in or linking accounts, try deleting browser cached files and cookies.

Google Chrome

  1. Select the Chrome menu icon on the top right and then select Settings.
  2. Near the bottom of the page, select Show advanced settings.
  3. Under Privacy, select Content settings.
  4. Under Cookies, either:
    • Enable all cookies: Select Allow local data to be set (recommended).
    • Enable pi.pearsoned.com: Select Manage exceptions..., and enter pi.pearsoned.com and set it to Allow. Select Done.
  5. Select Done and close the tab.

Firefox

  1. Select the Firefox menu icon on the top right and then select Options.
  2. Select Privacy from the left panel.
  3. Under History, set Firefox will to Use custom settings for history and either:
    • Enable all cookies: Select Accept cookies from sites, and then set Accept third party cookies to Always.
    • Enable pi.pearsoned.com: Select Exceptions.... In the Address box, enter pi.pearsoned.com and select Allow.
  4. Select Save Changes and close the tab.

Internet Explorer

  1. From Tools, select Internet Options.
  2. Select the Privacy tab.
  3. Under Settings, move the slider to enable the level of cookies you want.
  4. Select Sites. In the Address box, enter pi.pearsoned.com and select Allow. Select OK.
  5. Select OK and close the tab.

Safari

  1. From Safari, select Preferences.
  2. Select Privacy.
  3. Select to Always allow to enable all cookies. (Note that you can't enable cookies only for a specific website in Safari.)

Check Pearson Support or your browser help for more information or if your browser isn't covered in these instructions.
