¿Qué sucede después de que solicito mi nombre de usuario y contraseña?

Si tiene una cuenta Pearson, recibirá un correo electrónico con su nombre de usuario y un enlace para restablecer la contraseña. Si tiene varias cuentas, en el correo electrónico aparece la lista de todos sus nombres de usuario y enlaces independientes para restablecer las contraseñas.

Si no recibe un correo electrónico, siga las instrucciones del mensaje que reciba después de solicitar el restablecimiento de la contraseña.

Nota: Si va a vincular cuentas desde un sistema de gestión de aprendizaje (LMS), la página Restablecer contraseña se abrirá en una nueva ventana o pestaña. Después de restablecer la contraseña, cierre ambas ventanas o pestañas y vuelva a su LMS para continuar.
